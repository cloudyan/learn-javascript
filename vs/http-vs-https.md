# http vs https

- http1.0
- http1.1
- https
- http2
- http3

HTTP1.0定义了三种请求⽅法： GET, POST 和 HEAD⽅法

HTTP1.1新增了五种请求⽅法：OPTIONS, PUT, DELETE, TRACE 和 CONNECT
