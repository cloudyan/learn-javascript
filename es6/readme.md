# ES6 (ES2015)

ES6 的模块自动采用严格模式，不管你有没有在模块头部加上"use strict";

严格模式是 ES5 引入的，具体参看 ES5 部分

- let && const
- 解构赋值
- 扩展运算符
- 对象属性简写
- 箭头函数
- 函数参数默认值
- 模板字符串
- `Set` && `Map`
- `Symbol` 第七种原始数据类型, 表示独一无二的值
  - 前六种是：`undefined`、`null`、布尔值（Boolean）、字符串（String）、数值（Number）、对象（Object）
- `Promise`
- `Proxy`
- 类 Class
- 模块化
